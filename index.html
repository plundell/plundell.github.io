Paragast Web App

What is a web app?

A progressive web app (PWA) runs in your browser, but thanks to modern web technologies these work much like native apps:

  • Live on your home screen
  • Store data offline
  • Run in the background
  • Notify you of new messages/content

How to install:

  1. Allow Paragast to send notifications
  2. Install the background service
  3. Add Paragast to your home screen
  4. Reload page to run the app
